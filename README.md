# Awesome Kanban Board

Финальный проект на React + TypeScript.

## Стек технологий
* **React** (Функциональные компоненты, Hooks)
* **TypeScript** (Интерфейсы, типизация пропсов)
* **React Router** (Маршрутизация задач)
* **CSS** (Адаптивная верстка, БЭМ-подобные классы)

## Особенности проекта
* **Управление задачами**: создание новых задач в Backlog и перемещение их по цепочке до Finished.
* **LocalStorage**: сохранение состояния задач после перезагрузки страницы.
* **Адаптивность**: корректное отображение на мобильных устройствах (до 415px).
* **Детальная страница**: возможность редактирования описания каждой задачи.

## Как запустить
1. `npm install` — установка зависимостей.
2. `npm start` — запуск проекта в режиме разработки.
3. `npm run build` — сборка проекта для продакшена.
